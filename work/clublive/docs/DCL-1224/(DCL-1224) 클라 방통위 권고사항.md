## 일감 번호

[DCL-1224](https://www.notion.so/2b53482ee7c080738c2debe574fdfeb8?v=3b9c8445e9634c0e85d696caa6e836f3&source=copy_link)

## 목적 

1. 현재 클럽라이브 서비스 내에서 제거한 검색 기능을 방송통신위원회 권고 사항을 올바르게 적용해 복구하고자 함

## 내용

1. 부적절한 검색어 필터링 기능
	1. 기존 어드민 페이지에서 컨텐츠 관리 > 금지단어 페이지에 '검색 금지 단어' 탭 추가
		1. 금지어 등록
			1. 정상 등록
				1. '등록 되었습니다.'
			2. 중복 등록 확인
				1. '이미 존재하는 단어입니다.'
			3. 미 입력 후 등록 시도
				1. '등록할 금지단어를 입력해주세요'
		2. 금지어 검색
			1. 사용자가 입력한 단어와 일치 / 포함 ?
		3. 금지어 삭제
			1. 리스트에서 단어 클릭
				1. 삭제 의사 묻는 alert 출력
	2. 해당 금칙어 입력 시 자동으로 필터링 후 Toast
2. 영상물 신고 개선 사항
	1. 영상 신고 버튼 클릭 시, =='신고하기' 가 아닌 '관계 법령' 사항==^[검색 결과에 관계 법령을 보여주는 방식](페이지??)로 이동하게 변경
3. 불법 영상물 업로드 차단 시(차단 당한 사용자를 말하는 것인지?), 관계법령 안내 알림 표시 방식 구현 작업
